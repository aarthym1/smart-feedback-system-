Smart Feedback System
The Smart Feedback System is a full-stack web application that allows users to submit feedback along with their name, email, message, and rating. Submitted feedback is stored in MongoDB and displayed dynamically on the frontend.

Features:
User-friendly feedback form
Real-time feedback display
Backend API using Node.js & Express
MongoDB database integration
Clean UI with card-based layout
Axios-based API communication

Tech Stack:
Frontend:
React.js
Axios
Inline CSS (card-based UI)

Backend:
Node.js
Express.js
MongoDB
Mongoose
CORS
dotenv
